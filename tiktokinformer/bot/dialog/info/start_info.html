TikTok Informer Bot Привет! Это бот, который будет присылать тебе уведомления, когда у твоих любимых тиктокеров появятся новые видео. Использование: * Для добавления в свой список просто отсылай боту список имён (с приставкой @ - к примеру, @thekiryalife) * Для удаления из своего списка пришли боту список имён с приставкой - (минус), к примеру: - @thekiryalife @karna.val * Для просмотра своего списка пришли боту * (звёздочку) * Для остановки бота - команда /stop
